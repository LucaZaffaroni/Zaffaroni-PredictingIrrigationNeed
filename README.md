# Zaffaroni-PredictingIrrigationNeed
## Project Work on Machine Learning and Data Mining M - B2126

### Link to the kaggle competiton

https://www.kaggle.com/competitions/playground-series-s6e4/

### Submission File
For each id, you must predict a class label (Low, Medium, High) for the Irrigation_Need target.
The file should contain a header and have the following format:

id,Irrigation_Need
630000,Low
630001,High
630002,Low
etc.

### Dataset Description
The dataset for this competition was generated from a deep learning model trained on the Irrigation Prediction dataset. 
Feature distributions are close to, but not exactly the same, as the original.

#### Files  ->  train.csv 
The training set for train validation and test, with Irrigation_Need as target

#### Size  ->  116.04 MB

#### Type  ->  csv

### Evaluation
Submissions are evaluated on balanced accuracy between the predicted class and observed target.
